How to fetch the latest grants.gov XML

Use the latest public grants.gov XML extract, then load that file here with the picker above.

  1. Go to https://www.grants.gov/
  2. Go to https://www.grants.gov/xml-extract
  3. Select the latest dated .xml file.

Each link opens in a new tab so this session stays in place. Keep the XML outside the repo and load it directly from Downloads or another local folder.
